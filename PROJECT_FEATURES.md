# Page of Artists - 프로젝트 기능 설명서

## 📋 프로젝트 개요
Page of Artists는 아티스트가 자신을 홍보하고, 음악을 공유하며, 팬들과 소통할 수 있는 웹 플랫폼입니다.

## 🎨 리빌딩 완료 사항 (2025.01.03)
- ✅ 전체 다크 테마 적용 (블루/화이트 컬러 팔레트)
- ✅ 반응형 디자인 개선
- ✅ 모든 주요 페이지 리디자인 완료
- ✅ 공통 컴포넌트 (Header, Footer) 개선
- ✅ UI/UX 전반적인 개선

---

## 🎨 페이지별 기능 설명

### 1. 메인 페이지 (MainPage)
**경로**: `/main`

**주요 기능**:
- **장르 선택 버튼**: 사용자가 선호하는 음악 장르를 선택할 수 있는 팝업을 열 수 있습니다
- **아티스트 프로필 카드**: 2D 카드 형태로 아티스트 프로필을 탐색하는 페이지로 이동
- **아티스트 숏폼**: 3D 효과가 적용된 숏폼 형태의 아티스트 프로필 페이지로 이동

**UI/UX**:
- 큰 이미지 카드 두 개로 메인 기능에 접근
- 호버 시 상세 설명 표시
- 다크 테마 적용

---

### 2. 장르 선택 페이지 (GenreChoice)
**경로**: `/choice`

**주요 기능**:
- **장르 버튼**: 11가지 장르 (Pop, Band, Funk, Rock, Alternative, Ballade, Hiphop, Jazz, R&B, Metal, Country)를 선택할 수 있습니다
- **드래그 앤 드롭**: 장르 버튼을 드래그하여 선택 목록에 추가할 수 있습니다
- **선택한 장르 목록**: 우측에 선택한 장르들을 표시하고 삭제할 수 있습니다
- **장르 저장**: 선택한 장르를 서버에 저장하는 기능

**UI/UX**:
- 직관적인 드래그 앤 드롭 인터페이스
- 선택한 장르를 한눈에 확인 가능
- 다크 테마 적용

---

### 3. 아티스트 프로필 카드 페이지 (CarouselSection)
**경로**: `/carousel`

**주요 기능**:
- **3D 캐러셀**: 카드 형태로 아티스트들을 탐색할 수 있는 3D 캐러셀
- **좌우 네비게이션**: 버튼 클릭 또는 스페이스바로 카드 탐색
- **아티스트 정보**: 각 카드에 아티스트 이미지, 이름, 해시태그 표시
- **프로필 링크**: 카드 클릭 시 해당 아티스트의 마이페이지로 이동

**UI/UX**:
- 3D 회전 효과가 적용된 캐러셀
- 키보드 단축키 지원 (스페이스바)
- 부드러운 애니메이션 전환
- 다크 테마 적용

---

### 4. 아티스트 마이페이지 (AMP_*)
**경로**: `/amp_[아티스트명]` (예: `/amp_coldplay`)

**주요 기능**:
- **3D 카드 효과**: 마우스 이동에 따라 카드가 3D로 회전하는 인터랙티브 효과
- **음악 재생**: 음악 파일을 재생/일시정지/정지할 수 있는 플레이어
- **플랫폼 링크**: YouTube, Spotify, Instagram, SoundCloud 등 외부 플랫폼 링크
- **커뮤니티 게시판**: 아티스트 관련 커뮤니티 게시글 표시
- **아티스트 정보**: 이름, 장르, 소개글, 프로필 이미지 표시

**UI/UX**:
- 인터랙티브 3D 카드 디자인
- 반응형 레이아웃
- 다크 테마 적용

**지원 아티스트**:
- Coldplay
- Green Day
- Nirvana
- Oasis
- 5 Seconds of Summer
- YUNGBLUD
- Knox
- LANY
- MGK
- The 1975

---

### 5. 로그인/회원가입 페이지 (Login)
**경로**: `/login`

**주요 기능**:
- **로그인 폼**: 사용자 이름과 비밀번호로 로그인
- **회원가입 폼**: 새로운 계정 생성
- **폼 전환**: 로그인과 회원가입 폼을 쉽게 전환 가능

**UI/UX**:
- 깔끔한 폼 디자인
- 직관적인 사용자 경험
- 다크 테마 적용

---

### 6. 마이페이지 관리 (ArtistMyPageManage)
**경로**: `/artistmypagemanage`

**주요 기능**:
- **프로필 편집**: 아티스트 이름, 장르, 소개글 수정
- **이미지 업로드**: 프로필 이미지 업로드 및 변경
- **플랫폼 링크 관리**: YouTube, Spotify, Instagram, SoundCloud 링크 설정
- **커뮤니티 게시글 작성**: 커뮤니티에 게시글 작성 및 관리
- **음악 업로드**: 음악 파일 업로드

**UI/UX**:
- 편집 모드와 뷰 모드 전환
- 3D 카드 효과 (편집 모드가 아닐 때)
- 직관적인 편집 인터페이스
- 다크 테마 적용

---

## 🎯 기술 스택

### 프론트엔드
- **React**: 사용자 인터페이스 구축
- **React Router**: 페이지 라우팅
- **Three.js / React Three Fiber**: 3D 그래픽 및 애니메이션
- **Axios**: HTTP 클라이언트
- **CSS3**: 스타일링 및 애니메이션

### 백엔드 (참고)
- Node.js & Express
- MySQL
- AWS (EC2, S3, RDS)
